👴 VovôTinder 👵
Conectando corações na melhor idade.
O VovôTinder é uma plataforma de relacionamento desenvolvida exclusivamente para o público idoso. O projeto foca em acessibilidade extrema, eliminando barreiras digitais e promovendo a inclusão social através de uma interface intuitiva e simplificada.

🚀 O Projeto

Desenvolvido em tempo recorde para um Hackathon, este MVP (Produto Mínimo Viável) foca no "design para todos". Sabemos que o público sênior possui necessidades específicas de visão e coordenação motora, por isso, cada pixel foi pensado para facilitar a usabilidade.

🌟 Diferenciais de UX/UI para Idosos:

Tipografia Ampliada: Textos com leitura clara e alto contraste.
Áreas de Toque Generosas: Botões grandes para evitar cliques acidentais.
Navegação Híbrida: Suporte a gestos (swipe) e botões físicos de navegação.
Feedback Visual Imediato: Confirmações claras para cada ação realizada.

🛠️ Funcionalidades (CRUD Completo)

Visualizar (Read): Explore perfis em um carrossel interativo.
Cadastrar (Create): Entre para a comunidade através de um formulário simplificado.
Editar (Update): Altere suas informações e bio a qualquer momento.
Excluir (Delete): Total controle sobre seus dados e privacidade.

💻 Tecnologias Utilizadas

HTML5: Estrutura semântica e acessível.
CSS3: Estilização moderna com Flexbox e CSS Scroll Snap.
JavaScript (Vanilla): Lógica de manipulação de DOM e persistência.
LocalStorage: Armazenamento de dados no navegador (funciona 100% offline).
